## PIN configuration 

For PZEM-004 (V1/V2):

 Wemos Pin|GPIO|Component|PZEM-004 Signal
:-:|:-:|:-:|:-:
Tx|1|PZEM0XX Tx|Tx
Rx|3|PZEM0XX Rx|Rx

For PZEM-004 (V3):

 Wemos Pin|GPIO|Component|PZEM-004 Signal
:-:|:-:|:-:|:-:
Tx|1|PZEM0XX Tx|Tx
Rx|3|PZEM016 Rx|Rx
